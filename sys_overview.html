System Overview

Because system integration is always a priority

System Diagram

Our system primarily consisted of high-level software running on an Intel Euclid, low-level software running on an Arduino, and audio-related software running on a RasPi. All of the systems were powered off of a single 3S lipo with 5V voltage regulators in place where needed to provide logic power to the appropriate components. Below is a diagram of our full system with major power and signal connections between all of our components. Major code blocks are also indicated on the diagram to show how different components communicate with each other.
